## SIL Open Font License 1.1 简要说明

这两个字体都采用 SIL Open Font License 1.1（OFL-1.1），这是一个自由开源许可证，专门为字体设计。主要特点：

1. **自由使用**: 可以在任何媒介中使用，无论是印刷还是数字
2. **自由分发**: 可以与其他软件捆绑分发
3. **自由修改**: 可以修改字体并重新分发
4. **商业使用**: 完全允许商业使用
5. **嵌入许可**: 可以嵌入到软件和文档中

完整的许可证文本请访问: https://scripts.sil.org/OFL
